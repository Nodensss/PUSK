# Алгоритм пуска УПЭ

Статическое приложение для телефона: чек-лист, просмотр, критические правила, тренажёр и история попыток.

## Быстрый локальный просмотр

Откройте `index.html` двойным кликом.

## Проверка с телефона в одной Wi-Fi сети

Запустите:

```powershell
powershell -ExecutionPolicy Bypass -File .\server.ps1
```

В окне появится адрес вида `http://192.168.x.x:4173`. Откройте его на телефоне, подключённом к той же Wi-Fi сети.

## Деплой

Это обычная статическая папка. Можно загрузить весь каталог на Netlify, Vercel, GitHub Pages или любой статический хостинг.

Внутренняя навигация работает через `#/...`, поэтому приложение нормально открывается и на хостинге в подпапке.

Для Netlify уже добавлены `_redirects` и `netlify.toml`.

Для Vercel уже добавлен `vercel.json`.
